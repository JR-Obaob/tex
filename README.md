# tex
These files contain the sources of my LaTeX projects posted on my website.

- In order for my files to be compiled, you need [cjquines.sty](https://github.com/cjquines/tex/blob/master/cjquines.sty) and other packages being stated in the .tex file
- But in the meantime, cjquines.sty will not longer be my primary package and it will be replaced by my personal package sometime in the future. 
- For the figures and diagrams, I used [Mathcha](https://www.mathcha.io/editor)
